# React Template

A production-ready React starter template for shipping products fast. Built with modern tooling and a pre-built landing page.

**Repo:** [github.com/shashankaz/react-template](https://github.com/shashankaz/react-template)

## Stack

- **React 19** with React Compiler
- **TypeScript**
- **Vite 8**
- **Tailwind CSS 4**
- **shadcn/ui** + Radix UI primitives
- **Lucide React** icons
- **Recharts** for charts
- **Sonner** for toasts
- **Embla Carousel**
- **Geist** font

## Getting Started

```bash
git clone https://github.com/shashankaz/react-template.git
cd react-template
npm install
npm run dev
```

## Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Start development server |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm run format` | Format code with Prettier |
| `npm run format:check` | Check formatting without writing |
